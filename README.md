# Conecta Fatec Showcase

A static visual showcase of **Conecta Fatec**, an academic social network concept made for Fatec students.

This version is not functional yet. It was created to present the project idea, layout, pages, and navigation using only static files and mock data.

## About

Conecta Fatec is a social network concept focused on connecting students through profiles, posts, friendships, communities, and academic interactions.

All users, posts, comments, and communities in this repository are fictional and written directly in the HTML.

## Pages

The showcase includes:

- fake login page
- academic feed
- user profile
- friend profile
- communities page
- community pages
- friends page

## Technologies

- HTML
- CSS
- JavaScript
- Bootstrap
- Tailwind CDN

## Important note

This is only a front-end showcase.

There is no real login, backend, database, or data persistence. The buttons and actions are only visual, created to demonstrate how the final project could look.
